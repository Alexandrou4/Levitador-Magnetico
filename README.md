<h1 align="center"> **LEVITADOR MAGNÉTICO** </h1>

<h2> Nuestro proyecto, un levitador magnético por repulsión es un dispositivo que utiliza la fuerza de campos magnéticos para suspender un objeto en el aire sin contacto físico. </h2>

<h2> En este utilizamos los principios de electromagnetismo y corrientes eléctricas, PWM (Pulse Width Modulation/Modulación por Ancho de Pulso) y PID (Proporcional-Integral-Derivativo) como un sistema de control para regular constantemente la tensión media inducida en un electroimán en tiempo real y un circuito de potencia que nos permite tener la suficiente alimentación para el bobinado. </h2>

<h2> En este repositorio de Github se encuentran varias direcciones como documentación, archivos del firmware, hardware y del sitio web para el facil acceso del lector, con una breve exlicación de los contenidos de cada dirección </h2>
