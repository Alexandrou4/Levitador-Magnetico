Sistema de Control Aéreo Magnético

Tecnología de estabilización avanzada

Nariz: 125mm
Eje central: 105mm


Principio de Funcionamiento

El sistema utiliza dos sensores magnéticos para detectar la distancia a una superficie magnética. Cuando los sensores detectan diferencias, el sistema ajusta automáticamente la inclinación del avión para mantener la estabilidad.

Configuración de Sensores

Los controles deslizantes permiten simular diferentes condiciones de vuelo. El sensor de la nariz controla la inclinación frontal (pitch), mientras que el sensor del eje central controla el balanceo (roll).

Parámetros Técnicos

Rango de detección: 30-200mm. Precisión: ±1mm. Tiempo de respuesta: 50ms. Frecuencia de muestreo: 100Hz. Temperatura operativa: -20°C a 60°C.
